# sm-dashboard

A school management system dashboards.
Features includes:
- multiple login clients (admin, teacher, student, parent)
- multi-object relationships

## Technologies

Built with Node.js ecosystem

Frontend:
- Next.js
- Tailwind CSS
- various libararies for charts & forms

Backend:
- Prisma ORM
- Postgres database
